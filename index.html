Project Idea

    Overview

    The smarter connected home Hub..

    This project is to bring together many types of different smart technologies found in the home and place them all into a single hardware and software driven application rather than having to control them via the use of several different apps or devices. The smarter connected home should be streamlined to make life easier for the end user whilst integrating fully into one’s day to day life. At the same time automating processes around the home to work together seamlessly through machine learning and AI technology..

    Motivation

    Whilst we live in a world lined with technology these days it is not as cohesive as it could be. Currently your home may contain some smart lighting, an electronic door lock or a range of other smart driven devices, what would make the smart home even smarted would be if these devices could work together as a whole from a single app. Smart devices currently work mostly independent or with limited ability to connect to a single platform. Through the use of some fairly simple hardware design and software we could take control of these devices to work together..

    Description

    The smarter connected home hub and app would combine current smart devices along with other electronics that do not have smart technology and put them together in a single platform. .

    Starting with the front door, we can take a security camera and couple this with the use of machine learning services to identify faces or people and then integrate this with the electronic door lock so that as a person approaches the front door, they will be detected and the door will automatically unlock. Depending on the time of day this could then lead to turning on the house lighting in common areas and setting the inside temperature based on current weather conditions..

    If a familiar face is detected a notification can be sent to your phone or device which would ask if you would like to remotely unlock the door to let them in..

    The possibility would be endless with user customisable automations which could be programmed in a simple to use interface within an IOS or Android device. The smarter home app would have the ability to control television and home theatre using existing smart hardware and integrating with software..

    A geotag could be used to shut down all lights or electronics that are on if all users have left the house or specific electronics based upon the user. When everyone has left the house this could trigger the alarm system to activate and protect saving you the extra time and frustration of logging in to another piece of hardware..

    The hardware would be an expandable set of zone inputs and relay outputs that could control things such as electronic blinds, garage doors, outdoor lighting or sprinkler systems and the list of possibility is endless when it comes to devices that can be controlled via relay inputs. Zone inputs can be used to monitor the state of devices or sensors such as ambient light or motion..

    Having all of the technology working together on a single platform will also enable the smarter hub to be controlled via one of the already popular working voice activated smart assistants as opposed to individual smart devices connecting to these existing platforms directly. The difference would be that not all devices can be connected to one single platform as they are often designed with a single platform in mind. The smarter hub will support all devices with the ability to be controlled by voice via your smart assistant of choice..

    To summarize the smarter home hub would be an app controlled device with the capability of connecting all smart devices together along with other technologies and equipment to simplify the use of these devices through a single app and the use of smart automation without the need to run multiple apps and settings..

    Skills Required

    The skills required to make this project work are as follows:.

    This will require learning a programming language in order to program an application for a portable device or computer as well as writing a program that uses either the API if available from existing smart devices or custom script to interface. As well as firmware for the hub hardware..

    The hub would be a specially designed piece of hardware to control non smart enabled devices and will need skills in electronics to build..

    The hardware would not be too difficult to build however the software development will require a lot of knowledge and skill and would be undertaken by a team..

    Outcome

    If this project were to be successful this would eliminate the need for having to install multiple apps for controlling different smart devices and would allow control over equipment that traditionally would involve using physical switches or hardware to control. The project would take complex tasks and put them into an easy to use and understand single app that could be used by anyone no matter what age or with limited knowledge or technical ability..
